# :rose: Rose

_async_ Rust webservices

[![Project Status](https://img.shields.io/badge/status-in%20development-orange?style=for-the-badge)](https://github.com/CorentinLeGuen/nuage/tags)

## Run service

Service should be available at [localhost:8003/health](localhost:8003/health)

### Using cargo

`cargo build` to build or `cargo run` to run the service

### Using Docker

Build `docker build -t rose .` and run `docker run -p 8003:8003 rose`
