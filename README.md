# :rose: Rose

Rose (Rust Object storage Env) is a service based on `object_store` to keep track of space used.

[![Project Status](https://img.shields.io/badge/status-in%20development-orange?style=for-the-badge)](https://github.com/CorentinLeGuen/nuage/tags)

## Run service

Health service is available at [localhost:8003/health](localhost:8003/health)

### Using cargo

`cargo build` to build or `cargo run` to run the service
